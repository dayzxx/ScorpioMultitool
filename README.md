# Scorpio Multitool

> A powerful OSINT \& network multitool made by n0pt



\---



\## Features



| # | Tool | Description |

|---|------|-------------|

| 1 | IP Lookup | Get detailed info about any IP address |

| 2 | Webhook Sender | Send messages to a Discord webhook |

| 3 | IP Pinger | Continuously ping an IP or check a port |

| 4 | Port Scanner | Scan common or custom port ranges |

| 5 | URL Shortener | Shorten any URL via TinyURL |

| 6 | Username Lookup | Search a username across 26 platforms |

| 7 | Metadata Scanner | Extract EXIF/PDF metadata from files |

| 8 | DNS Lookup | Resolve a domain to its IP address |

| 9 | Whois Lookup | Get registration info about a domain |

| 10 | Phone Lookup | Get info about a phone number |

| 11 | Reverse Image Search | Search an image on Google Lens, TinEye \& Yandex |



\---



\## Installation



\*\*Requirements:\*\* Python 3.10+



1\. Clone the repo

```

git clone https://github.com/dayzxx/ScorpioMultitool.git

cd ScorpioMultitool

```



2\. Run the setup

```

python setup.py

```



3\. Launch the tool

```

python MultiTool.py

```
