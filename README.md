# Scorpio Multitool
> A powerful OSINT & network multitool made by dayzx

---

## Features

- **IP Lookup** — Get detailed info about any IP address
- **Webhook Sender** — Send messages to a Discord webhook
- **IP Pinger** — Continuously ping an IP or check a port
- **Port Scanner** — Scan common or custom port ranges
- **URL Shortener** — Shorten any URL via TinyURL
- **Username Lookup** — Search a username across 26 platforms
- **Metadata Scanner** — Extract EXIF/PDF metadata from files
- **DNS Lookup** — Resolve a domain to its IP address
- **Whois Lookup** — Get registration info about a domain
- **Phone Lookup** — Get info about a phone number
- **Reverse Image Search** — Search an image on Google Lens, TinEye & Yandex

---

## Installation

Requirements: Python 3.10+

```
git clone https://github.com/dayzxx/ScorpioMultitool.git
cd ScorpioMultitool
python setup.py
```
